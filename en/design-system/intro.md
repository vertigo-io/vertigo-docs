# Vertigo Design System

Vertigo's ambition is to build digital services faster and more efficiently.

For web applications, user interfaces are fundamental, and having a Design System provides the means to achieve this goal.

We designed and built this Design System with Vertigo's philosophy: simplicity, robustness, security, scalability.

Vertigo is an _opinionated_ solution, meaning it defends specific principles and targets particular use cases, especially business applications. This Design System shares the same characteristics.
The choices we made will allow you to reach your goals faster with a higher final quality level.

The principles that guided the creation of this Design System are:

- clarity of user interfaces
- application reliability
- application security
- accessibility
- smartphone compatibility

To document this Design System, we chose to use the Atomic Design formalism, which is organized into several levels: atoms, molecules, organisms, and templates.
These different levels compose successively to create pages.
