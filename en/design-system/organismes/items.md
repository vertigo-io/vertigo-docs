# Item List

The item list is somewhat an intermediate between a table and a card.
It is a good choice when you want to display graphic content but the information is predominantly textual.

This format adapts well to mobile in certain cases.

# Best Practices

- Limit the number of available actions on a line (<3); for complex actions, prefer navigation to a dedicated screen
