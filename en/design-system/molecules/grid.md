# Grid

To densify input and view screens, we provide a grid component.
It allows placing multiple input fields on the same line across different columns, making advantageous use of CSS3 capabilities.

This component supports responsive web design and adapts to different screen sizes.
Thus, you can have a 2 or 3 column display on a computer and automatically switch to a single column display on a mobile phone.

It also allows mixing a variable number of columns within the same grid for greater flexibility.

# Best Practices

- The two-column mode should be preferred and is the default value
- Densification must not override clarity and consistency. Avoid making grids too large (> 8 rows) and instead use blocks to structure information.
