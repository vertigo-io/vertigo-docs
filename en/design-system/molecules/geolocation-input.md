# Geolocation

As a rule, location information is provided by a hardware device or via a mapping tool, and manual input of this information remains exceptional.
However, it is not uncommon to need to visualize and correct this information.

# Best Practices

- A map is the ideal choice for visualizing this type of information.
- The input of cartographic information is a domain in itself. It would be complex to standardize. We therefore prefer to keep these choices within the scope of each project.
