# Prerequisites

In this guide, we will help you create your first Vertigo application, in two steps:
* How to create a minimal application and start it
* How to create a real web application, with screens, a relational database, business services, etc...


To follow this guide, you will need:
* Java Development Kit properly installed - Version: __JDK 17 or higher__
* __Maven (version 3.6+ or higher)__ or __Gradle (version 7.3+ or higher)__ - Maven will be used for this guide)
* A text __editor__ or better, a Java __IDE__ (we recommend the Eclipse IDE, which will be used in this guide)
